# ZUKKO — Intellektual Bilimlar Jangi 🚀

**ZUKKO** — bu O'zbekistondagi eng zamonaviy va qiziqarli intellektual viktorina platformasi. Foydalanuvchilar o'z bilimlarini turli fanlar bo'yicha sinab ko'rishlari, ballar to'plashlari va noyob tarixiy artefaktlar to'plamini yig'ishlari mumkin.

## ✨ Asosiy Xususiyatlar

- **Fanlar bo'yicha testlar**: Matematika, Tarix, Informatika, Biologiya va boshqa ko'plab fanlar.
- **Xazina (Collection) tizimi**: O'yin davomida noyob tarixiy artefaktlarni (masalan, Ulug'bek astrolyabiyasi, Amir Temur qilichi) topish va to'plash imkoniyati.
- **Reting va Ranhlar**: "Talaba"dan "Professor"gacha bo'lgan yo'lni bosib o'ting.
- **Bozor va Avatarlar**: To'plangan ballar evaziga yangi avatarlar va yordamchi buyumlar (yuraklar, muzlatish) sotib oling.
- **Xatolar ustida ishlash**: Yutqazilgan savollarni qayta ko'rib chiqish va bilimlarni mustahkamlash tizimi.

## 🛠 Texnologiyalar

- **Frontend**: React.js + Vite
- **Dizayn**: Premium Glassmorphism UI & Custom CSS Animations
- **Backend/Database**: Supabase (Realtime data & Secure points)
- **Deployment**: Vercel

## 🚀 Ishga tushirish

Loyiha bilan tanishish uchun:

1. Repozitoriyani klonlang: `git clone https://github.com/USER_NAME/zukko-app.git`
2. Kutubxonalarni o'rnating: `npm install`
3. Loyihani ishga tushiring: `npm run dev`
